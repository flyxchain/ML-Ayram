# ML-Ayram — Trading Bot con ML para Forex

Sistema de trading algorítmico con Machine Learning para análisis de pares de divisas,
generación de señales por Telegram y ejecución automática en cTrader.

## Documentación
- `docs/TUTORIAL_COMPLETO.md` — Tutorial paso a paso de todo el proyecto
- `docs/PROMPT_CONTINUIDAD.md` — Prompt para continuar el proyecto en nuevos chats con Claude

## Stack
Python 3.11 · XGBoost · TensorFlow/Keras · PostgreSQL · cTrader API · Telegram Bot · Netlify

## ⚠️ Importante
Nunca subas el archivo `.env` a Git. Siempre prueba en cuenta DEMO antes de usar dinero real.
